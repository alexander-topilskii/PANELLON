# Открытые вопросы

На текущий момент блокирующих открытых вопросов перед стартом реализации нет.

## Закрытые вопросы (кратко)

| Вопрос | Ответ |
|---|---|
| Бесконечность или 9999 этажей? | Для v1 — потолок `9999`, архитектура готовится к practically-infinite chunking. |
| Как устранить конфликт по лестницам? | Этаж 0: вверх; этажи 1-5: два конца коридора; этажи 6+: две площадки (вверх/вниз) в одной точке, позиция по hash(floor). |
| Как не взорвать производительность на высоких этажах? | Только lazy chunked generation, без полной генерации этажа. |
| Что считается детерминизмом? | Детерминированы topology/Room_ID/grammar/cache-status. Битовая идентичность картинки на разных GPU не гарантируется. |
| Что делать с невалидными/черными комнатами? | Retry до лимита, затем `empty`; compile error ведет во fallback shader. |
| Как хранить и версионировать данные? | localStorage для resume, IndexedDB (`cache`, `archive`, `meta`), versioned cache keys. |
