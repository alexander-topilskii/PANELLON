# PANELLON

## Что это за проект

**PANELLON** — веб-приложение (SPA), в котором вы исследуете бесконечный 3D-лабиринт в стиле лиминальных пространств и «Гигахрущёвки».

Это **детерминированный цифровой монумент**: один и тот же сид всегда даёт один и тот же мир. Есть дефолтный «официальный» мир.

### Визуальный стиль

- **Лиминальные пространства** — пустые коридоры, подъезды, переходы без людей
- **«Гигахрущёвка»** — бесконечный многоквартирный дом в духе советской архитектуры

---

## Концепция: «Вертикальный Архив Энтропии»

Бесконечная Гигахрущёвка, растущая вверх. Весь мир строится на основе **Seed-фразы** (зерна генерации).

**Детерминизм:** два человека с одним сидом видят идентичные комнаты, шейдеры и коридоры.

---

## Навигация и управление

- **WASD** — движение по коридорам
- **Лестницы** — автогенерируемые, ведут между этажами

---

## Геометрия и рендеринг

- **Коридоры** — обычная геометрия (меши)
- **Комнаты** — Raymarching: вся комната является искажённым пространством, шейдер рисует геометрию

---

## Структура здания

- **Форма** — перевёрнутая пирамида
- **Планировка этажа** — сетка комнат
- **Сложность шейдера** — привязана к высоте (координата Y)

### Иерархия этажей

| Этажи | Название | Описание |
|-------|----------|----------|
| **1–5** | Пустота (Технический этаж) | Символьный лимит слишком мал для работающего шейдера. Лиминальная эстетика: голые стены, провода, тусклые лампы, звуки пустого здания. |
| **6–50** | Зарождение | Короткие шейдеры. Градиенты, мигающие цвета, шум. |
| **500+** | Фрактальный коллапс | Длинные шейдеры. Математические галлюцинации, пространство сворачивается само в себя. |

---

## Генерация шейдеров

- **Метод** — перебор символов (латиница, цифры, пробел, символы, допустимые в GLSL)
- **Связь с этажом** — количество/диапазон символов зависит от этажа
- **Проверка** — при движении идёт перебор и проверка компилируемости
- **Успех** → можно зайти в комнату, результат кэшируется (не «майнить» повторно)
- **Провал** → коридор без двери (зайти нельзя)

### Room ID

```
Room_ID = hash(Floor_Number + Room_Position + Global_Seed)
```

Room_ID — seed для генератора шейдеров. Генерация детерминирована.

---

## Система фильтрации и сохранения

### Авто-валидация

Если шейдер не компилируется или даёт чёрный экран — **мутация** (замена пары символов), пока не получится рабочий вариант.

### Заархивировать

Кнопка в интерфейсе. Сохраняет код шейдера + метаданные (комната, этаж) в IndexedDB. Планируется онлайн-галерея «лучших находок».

---

## Производительность

- Рисуем **одну комнату** за раз
- При входе на этаж — ожидание генерации и проверки

---

## Vision (кратко)

SPA с бесконечным процедурным 3D-лабиринтом. Коридоры — меши, комнаты — Raymarching с процедурно сгенерированными GLSL-шейдерами. Сложность растёт с высотой. Детерминированность по Seed-фразе.
