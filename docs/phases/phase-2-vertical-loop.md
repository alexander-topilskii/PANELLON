## 2.1 Этажи 0 и 1–5 (Специальные шаблоны)

- [ ] Определить модель Floor (`number`, `type: 'lobby' | 'linear' | 'procedural'`)
- [ ] **Этаж 0:** вестибюль 6×6×2.5 м (переиспользуем комнату из фазы 1), светящаяся площадка-лестница у дальней стены
- [ ] **Этажи 1–5:** прямой коридор 30×4×2.5 м (меши пола, потолка, стен), лестницы на обоих концах
- [ ] HUD-элемент «Этаж N» в углу экрана
- [ ] `FloorManager`: загрузка и выгрузка геометрии этажа (`scene.remove` → `dispose`)

## 2.2 Лестницы и fade-переходы

- [ ] Геометрия лестницы: PlaneGeometry (emissive material) + стрелка (вверх/вниз цветом)
- [ ] Триггер-зона: `Box3` intersection игрока и площадки
- [ ] При триггере: блокировка управления → fade-out 0.3 с (CSS overlay) → dispose старого этажа → генерация нового → спавн у лестницы → fade-in
- [ ] Позиции: этаж 0 — одна лестница вверх; этажи 1–5 — две лестницы на концах коридора
