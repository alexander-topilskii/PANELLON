# Фаза 7 — Продуктовый слой [M6]

**Цель:** Довести "песочницу" до полноценного цифрового продукта-монумента. Дать игроку возможность "архивировать" красивые генерации и сохранять прогресс.

## 7.1 Сохранение позиции (Resume)
**Что делаем:**
- В `localStorage` по ключам `panellon.seed`, `panellon.floor`, `panellon.position_x`, `panellon.position_z` сохраняем состояние каждые N секунд или при переходе между чанками/этажами.
- При загрузке страницы:
  - Если в URL есть `?seed=...`, используем его и стартуем с 0 этажа.
  - Если URL пуст, читаем `localStorage`. Если есть сохраненный сеанс — восстанавливаем игрока точно в те же координаты `(x, z)` на нужном этаже.
- UI стартового экрана модифицируется: появляется кнопка "Продолжить (Seed: XY, Этаж N)".

## 7.2 Полировка (Архив энтропии)
**Что делаем:**
- **Кнопка «Заархивировать»:** В режиме комнаты, когда игрок не двигает камеру (или по нажатию Escape), появляется UI кнопка.
- При клике -> шейдер + метаданные (этаж, координаты, дата) сохраняются в IndexedDB в store `archive`.
- `archive` никогда не очищается автоматически (в отличие от `cache` с его LRU-эвикцией).
- **Меню Архива:** На стартовом экране или по Escape добавляем раздел "Мой Архив", где можно посмотреть список сохраненных находок.
- При клике на элемент архива -> игрок телепортируется прямо в эту комнату (или просматривает ее как скриншот/мини-рендерер, но лучше — телепорт, чтобы "вернуться").

## 7.3 Fallbacks и UX
**Что делаем:**
- **WebGL Fallback:** Проверка на старте `WebGL.isWebGL2Available()`. Если нет — показываем красивый экран с ошибкой "Ваше устройство не поддерживает Вертикальный Архив".
- **Смена версии грамматики:** Жестко прописываем константу `GEN_VERSION` в коде. При ее смене старый кэш (но не архив!) сбрасывается, так как комнаты будут генерироваться иначе.
- **Предзагрузка (Prewarm):** Логика генерации комнат смещается. При спавне на этаже мы асинхронно предгенерируем комнаты в радиусе видимости (например, 15 ячеек).

**MVP (Definition of Done):**
- Игрок доходит до 12 этажа, закрывает вкладку.
- Открывает сайт завтра -> появляется кнопка "Продолжить", он появляется на 12 этаже в том же месте.
- Находит красивую комнату -> жмет "Заархивировать".
- В меню видит эту комнату, может перепройти к ней по нажатию.
- Готовый к публикации MVP проекта.
